# Image Assets for Quantity Survey App

This directory contains image assets for the Quantity Survey application.

## Icon Files (SVG format recommended):
- app-icon.svg - Main app icon (32x32, 64x64, 128x128)
- boq-icon.svg - Bill of Quantities icon
- valuation-icon.svg - Valuation icon
- variation-icon.svg - Variation Order icon
- payment-icon.svg - Payment Certificate icon
- measurement-icon.svg - Measurement icon
- dashboard-icon.svg - Dashboard icon
- report-icon.svg - Report icon

## Image Files:
- logo.png - Company/App logo
- construction-bg.jpg - Background image for construction projects
- measurement-tool.png - Measurement tool illustration
- mobile-app-preview.png - Mobile app preview image

## Usage:
Place actual image files in this directory following the naming convention above.
Update the paths in templates and JavaScript files accordingly.

## Recommended Sizes:
- Icons: 16x16, 24x24, 32x32, 48x48, 64x64, 128x128 pixels
- Logo: 200x60 pixels (landscape) or 120x120 pixels (square)
- Background images: 1920x1080 pixels
- Illustrations: 400x300 pixels

## File Formats:
- Icons: SVG (preferred) or PNG
- Logos: SVG or PNG with transparency
- Photos: JPG or WebP
- Illustrations: SVG or PNG
